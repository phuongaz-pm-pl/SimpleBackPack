# SimpleBackPack

Simple BackPack for Pocketmine-MP

#### Example plugin for using [ItemHolder](https://github.com/phuongaz-pm-pl/ItemHolder) and [AzEconomy](https://github.com/phuongaz-pm-pl/AzEconomy)

#### Features:
- Support SQLite
- [pmforms](https://github.com/dktapps-pm-pl/pmforms) support
- [AzEconomy](https://github.com/phuongaz-pm-pl/AzEconomy) support
- Upgrade backpacks (Chest -> Double chest)
